💰 Feature 2: Off-Ramp (Stablecoin to Fiat)

Send stablecoin and receive fiat in a bank account. Two approaches: First Party or Third Party.

First Party Off-Ramp (Receive in Own VA)

User receives fiat in their own Accounts API Virtual IBAN.

Why use this:

  • User wants fiat in Bakkt custody
  • Compliance - fiat in user's own name
  • Can use fiat for other Bakkt services

Requirements:

  • ✅ Accounts API Virtual IBAN in user's name
  • ✅ Link VA as beneficiary in On/Off Ramp

Note: This section includes a sequence diagram showing the workflow. Refer to the original documentation for the visual diagram.

User Example:

// Prerequisites: User has Accounts API Virtual IBAN

// 1. Add own VA as beneficiary in On/Off Ramp
const beneficiary = await fetch('https://api.bakkt.com/user/bank-account/remote', {
  method: 'POST',
  headers: {
    'Authorization': apiKey,
    'unblock-session-id': sessionId
  },
  body: JSON.stringify({
    account_name: 'My Bakkt Account',
    main_beneficiary: true,
    account_details: {
      currency: 'EUR',
      iban: paymentsVA.iban  // IBAN from Accounts API
    }
  })
});

// 2. Get wallet address
const wallet = await fetch('https://api.bakkt.com/user/wallet/polygon', {
  headers: {
    'Authorization': apiKey,
    'user-uuid': userUuid
  }
});

// 3. User sends USDC to wallet address
// → Auto-converts to EUR
// → EUR sent to user's own Payments VA
// → User can hold, withdraw, or use for other services

Corporate Example:

// 1. Link corporate's Payments VA as beneficiary

GET /corporate/{corporate_uuid}/wallet/polygon

// Corporate sends crypto → Fiat to own VA

Third Party Off-Ramp (Send to Any Bank)

Send fiat to any external bank account - no Accounts API needed.

Why use this:

  • Paying suppliers or partners
  • Remittance to family/friends
  • No need for recipient to have Bakkt account

Requirements:

  • ✅ External bank account details only
  • ❌ Does NOT require Accounts API

Note: This section includes a sequence diagram showing the workflow. Refer to the original documentation for the visual diagram.

User Example:

// No Accounts API needed for third party!

// 1. Add external beneficiary bank account
const beneficiary = await fetch('https://api.bakkt.com/user/bank-account/remote', {
  method: 'POST',
  headers: {
    'Authorization': apiKey,
    'unblock-session-id': sessionId
  },
  body: JSON.stringify({
    account_name: 'Supplier Payment',
    main_beneficiary: false,
    account_details: {
      currency: 'EUR',
      iban: 'FR1420041010050500013M02606'  // Any external IBAN
    }
  })
});

// 2. Get wallet address
const wallet = await fetch('https://api.bakkt.com/user/wallet/polygon', {
  headers: {
    'Authorization': apiKey,
    'user-uuid': userUuid
  }
});

// 3. User sends USDC to wallet
// → Auto-converts to EUR
// → EUR sent to external bank account

Corporate Example:

// Pay suppliers without Accounts API

POST /corporate/{corporate_uuid}/bank-account/remote
{
  "account_name": "Supplier ABC",
  "account_details": {
    "currency": "USD",
    "account_number": "123456789",
    "bank_code": "026009593",
    "bank_name": "Bank of America"
  }
}

Warning: Key Difference:

  • First Party: Requires Accounts API VA (fiat stays in Bakkt ecosystem)
  • Third Party: Standalone (fiat leaves to external account)

Complete Offramp Example

// Step 1: Add destination bank account
const bankAccount = await bakkt.addDestinationAccount(sessionId, {
  account_name: 'My Main Account',
  main_beneficiary: true,
  account_details: {
    currency: 'EUR',
    iban: 'DE89370400440532013000'
  }
});

console.log('Bank account UUID:', bankAccount.uuid);
console.log('Crypto receiving address:', bankAccount.receiving_address);

// Step 2: Get wallet address (optional - can use receiving_address above)
const wallets = await bakkt.getWallet(sessionId, 'polygon');
console.log('General wallet:', wallets[0].address);

// Step 3: User sends crypto to receiving_address
// (Happens in user's wallet app)

// Step 4: Monitor transaction
webhook.on('PROCESS_COMPLETED', (data) => {
  if (data.direction === 'cryptoToFiat') {
    console.log('Offramp complete!');
    console.log(`${data.input.amount} ${data.input.currency} converted to`);
    console.log(`${data.output.amount} ${data.output.currency}`);
    console.log('Fiat sent to bank account');
  }
});
