Overview

The Onboarding APIs are your entry point to the Bakkt platform. It handles individual user and corporate entity creation, authentication, identity verification, and merchant configuration.

Explore core features and endpoints

User Management

Create and manage individual user accounts with comprehensive profile data.

Core endpoints

POST /user Create a new user account with basic information and crypto addresses

PATCH /user Update user profile details, addresses, and preferences

GET /user Retrieve complete user profile including status and verification level

Core endpoints

Authentication

Authenticate users via SIWE (Sign In With Ethereum) or Email OTP.

View Full Authentication Guide →

Core endpoints

POST /auth/login/ Authenticate via SIWE or trigger Email OTP

POST /auth/otp/ Validate OTP and receive session ID

Common Use Cases

Create and authenticate a user

sequenceDiagram
participant App as Your App
participant Bakkt as Bakkt API
participant User as End User
App->>Bakkt: POST /user (create user)
Bakkt-->>App: user_uuid
App->>Bakkt: POST /auth/login (trigger OTP)
Bakkt->>User: Email with OTP
User->>App: Enters OTP
App->>Bakkt: POST /auth/otp/ (submit OTP)
Bakkt-->>App: session_id
App->>Bakkt: Authenticated requests with session_id

Corporate Onboarding

sequenceDiagram
participant App as Your App
participant Bakkt as Bakkt API
participant Admin as Corporate Admin
App->>Bakkt: POST /corporate (create entity)
Bakkt-->>App: corporate_uuid
App->>Bakkt: POST /corporate/{uuid}/user (add admin)
App->>Bakkt: GET /corporate/{uuid}/kyb/applicant/url
Bakkt-->>App: KYB form URL
App->>Admin: Send KYB form link
Admin->>Bakkt: Completes KYB (external form)
Bakkt-->>App: Webhook: KYB approved

Integration Example

Complete example of user onboarding flow:

// 1. Create user
const user = await bakkt.createUser({
  first_name: 'Alicekkkk',
  last_name: 'Smith',
  email: 'alice@example.com',
  country: 'GB',
  accepted_terms_of_service_hash: 'base64_hash_here'
});

// 2. Authenticate
await bakkt.triggerOTP(user.user_uuid);
const session = await bakkt.submitOTP(user.user_uuid, '123456');

// 3. Initiate KYC
await bakkt.createKYCApplication(session.session_id);
const sumsubToken = await bakkt.getKYCToken(session.session_id);

// 4. User completes KYC in your UI
// (Using Sumsub Web SDK with the token)

// 5. Monitor KYC status via webhook or polling
const kycStatus = await bakkt.getKYCStatus(session.session_id);

// 6. Once approved, user can transact
if (kycStatus.status === 'FULL_USER') {
  // User ready for payments!
}
